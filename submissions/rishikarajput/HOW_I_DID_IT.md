## How I did it
I forked and cloned the repository, then set up the project with npm install and npm run build. I ran npm run test-client to check everything, I expected something to break, but all 8 tests passed on the first try.
## Problems I faced
The biggest hurdle early on was just knowing where to start. The README ended up being more helpful than I gave it credit for initially. Ollama was straightforward to install, though i rechecked my wifi, becuase it was so slow.
## What I learned
The main thing I walked away with was a clearer picture of how local LLMs actually work in practice — not just as a concept but as something you can run, query, and build on top of. Setting up the Node.js pipeline from scratch also made npm scripts feel a lot less mysterious.
