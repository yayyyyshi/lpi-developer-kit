## HOW I DID IT 
I started by forking the repo and cloning it locally, then ran npm install and npm run build to get everything set up. Honestly, I wasn't sure if it worked at first, but running npm run test client cleared that up pretty fast , all 8 tests passed, which was a relief. The trickiest part was just figuring out the initial setup since the README wasn't immediately obvious to me, but once I followed it properly things clicked into place.
## PROBLEMS FACED
For Ollama, the setup itself was simple enough, just the model download took forever. Once llama3 was running though, it was genuinely cool to see a language model running entirely on my own machine.
## WHAT I LEARNED 
Looking back, I got more comfortable with npm scripts and Node.js project structure than I expected. What actually stuck with me was understanding how the LPI sandbox tools connect together ,it's one thing to read about tool use in LLMs, it's another to watch it work in a test environment you set up yourself.
