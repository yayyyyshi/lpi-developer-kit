This project implements an AI agent that provides health advice using structured knowledge from LPI tools and the SMILE methodology.

## LPI Tools Used
- smile_overview

---

## Example Input
"I feel tired and not sleeping well"

---

## Example Output
The agent analyzes the problem and suggests:
- Improve sleep duration
- Maintain consistent routine
- Improve nutrition and hydration

---

## Explainability
The agent works in 3 steps:
1. Understands user query
2. Fetches structured knowledge from LPI tools
3. Combines results into actionable advice

The reasoning is transparent and based on known health principles.

---

## Why This Is Useful
This agent helps users understand health issues using structured reasoning instead of generic responses.
